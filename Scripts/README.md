#### _Silent Install applications on linux_
# Ubuntu-Script
This script when it is executable makes the silent install of the programs mentioned in the README without any user intervention
**_This file works with most of debian based distribution like Zorin, Linux Mint etc_**
# Programs are:-
1. JDK 11
2. Chrome
3. Sublime Text
4. VS Code
5. Telegram
6. VLC
7. Eclipse
8. Netbeans
9. Miniconda follow this:- [link](https://varhowto.com/install-miniconda-ubuntu-20-04/)
10. Spotify
11. Python3 Virtual Environment
12. Python3 pip
13. For node follow:-[link](https://www.youtube.com/watch?v=JS2aIYd5ADI) 
# Installation process:
### Extract the file  
### Open terminal in the same location as file and execute 
>chmod +x ubuntu-install-programs.sh 
### Use below command to execute it
> bash ubuntu-install-programs.sh
